
Generate a 16 measure loop of four chords (I, IV, vi, V) in D major.

Arguments:
  outfile:  Name of file to be written, must be .wav.
  waveform: Type of waveform to generate, one of:
              - pulse
              - sawtooth
              - sine
              - square
              - triangle
  bpm:      Desired tempo in beats per minute, in range 1-300.
  volume:   Loudness of generated waveform, from 1-11.
